# FORK()   
- Calculate the number of ```Fork()``` processes with -> $2^n=X$
- Example: 
     ```cpp  
     int main(int argc, char* argv[]){
        fork();
        fork();
        for();
        fork();
        printf("Hello, World);
        return 0;
        }
    ```

- $2^4=16$
